sVimPy by pez2001

a very small python virtual machine, intended for use in microcontroller projects

this project aims to have a very small memory footprint. still at alpha stage so dont expect a fully working interpreter at the moment. class support is missing and type are only supported rudimentary.

Dependencies

no dependencies. use gcc to compile the sourcecode
